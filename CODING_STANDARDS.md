# Coding standards

## Commits

- **AI-authored commits use a `Generated-by:` trailer, never `Co-Authored-By:`.**
  An AI assistant is a tool, not a co-author, so it must not be credited as one.
  End such commit messages with a trailer like:

  ```
  Generated-by: Claude Opus 4.8
  ```

  Do not add `Co-Authored-By:` lines for AI assistants under any circumstance.
